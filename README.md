# React 

This repository follows the share knowledge meetups to build React from scratch in small steps. it shows how to build a small counter app using React architecture without any third party library.

## Installation

```bash
pnpm install
```

## Run

```bash
npm run dev
```

## Build

```bash
npm run build
```
